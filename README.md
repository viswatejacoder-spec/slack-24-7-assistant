# StarDance Assistant 🤖

A 24/7 Slack AI Agent built using Node.js and Slack Bolt.

## Features
- Responds to Slack messages
- Remembers user conversations
- Supports slash commands
- Provides conversation history
- Runs continuously using Slack Socket Mode

## Commands
- /about
- /help
- /history

## Technologies Used
- Node.js
- Slack Bolt SDK
- JavaScript
- GitHub

## Installation

1. Clone the repository
2. Run `npm install`
3. Add environment variables in `.env`
4. Run `npm start`
